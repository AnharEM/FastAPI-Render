# FastAPI Render

A simple FastAPI application built with **FastAPI** and **Pydantic** for data validation and API development.

## Features

- FastAPI-based REST API
- Request and response validation using Pydantic
- Automatic API documentation
- Lightweight and easy to extend

## Requirements

- Python 3.9+
- FastAPI
- Uvicorn
- Pydantic
